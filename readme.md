## 🎯 프로젝트 개요

### 🔥 핵심 아이디어
**"리스크를 적으로 여기지 말고, 언락해야 할 기회로 바라보자"**

Walk Risk는 투자의 리스크를 게임화된 방식으로 체험하고 정복하는 혁신적인 교육 플랫폼입니다. 실제 시장 데이터와 연동하여 안전한 환경에서 리스크 관리 능력을 기를 수 있습니다.

### 🌟 프로젝트 특징
- **📊 실시간 시장 데이터 연동**: Yahoo Finance API를 통한 실제 시장 반영
- **🎮 완전한 게임화**: 레벨업, 경험치, 아이템, 도전과제 시스템
- **🇰🇷 한국 시장 특화**: 삼성전자, 코스피 등 한국 투자 환경 반영
- **👨‍🏫 AI 멘토 시스템**: 워렌 버핏, 피터 린치 등 유명 투자자 AI 가이드
- **💰 단계별 실전 투자**: 10만원부터 30억원까지 체계적 성장

## 💡 핵심 철학: 리스크 언락

### 🔓 리스크의 재정의
```
전통적 관점: Risk = Danger (위험)
언락 관점: Risk = Locked Opportunity (잠긴 기회)

리스크 상태:
├── 🔒 Locked Risk    - 아직 이해하지 못한 위험
├── 🔓 Unlocking Risk - 분석하며 이해하는 중
├── 🔑 Unlocked Risk  - 정복하여 관리 가능한 상태  
└── 💎 Mastered Risk  - 기회로 전환한 완전 정복
```

### 🌟 교육 철학
1. **실패는 성장의 재료** - 모든 손실에서 배우는 시스템
2. **실전이 최고의 스승** - 이론보다 경험 중심 학습
3. **단계적 성장** - 작은 성공을 쌓아가는 점진적 발전
4. **커뮤니티 학습** - 함께 배우고 성장하는 소셜 시스템

### 🔑 리스크 언락 시스템
각 투자 리스크는 게임의 '던전'과 같습니다. 플레이어는 적절한 '키'를 수집하여 리스크를 언락하고 마스터합니다.

```python
# 리스크 언락 예시
class RiskChallenge:
    def __init__(self, risk_name, required_keys, reward):
        self.name = risk_name
        self.keys_needed = required_keys  # ['knowledge_key', 'experience_key']
        self.reward = reward
        
    def attempt_unlock(self, player_keys):
        if self.check_keys(player_keys):
            return self.grant_mastery()
        return "더 많은 경험이 필요합니다!"

# 실전 사용
volatility_risk = RiskChallenge(
    risk_name="변동성 마스터",
    required_keys=['chart_analysis', 'loss_experience', 'patience'],
    reward={'exp': 500, 'title': '볼린저 밴드 마스터'}
)
```

```python
# 실시간 시장 데이터 연동 예시
import yfinance as yf
from datetime import datetime

class MarketDataIntegrator:
    def get_risk_level(self, ticker):
        stock = yf.Ticker(ticker)
        data = stock.history(period="1mo")
        
        # 변동성 계산
        volatility = data['Close'].pct_change().std() * 100
        
        if volatility > 5:
            return "HIGH_RISK"
        elif volatility > 2:
            return "MEDIUM_RISK"
        else:
            return "LOW_RISK"

# 삼성전자 리스크 레벨 확인
samsung_risk = MarketDataIntegrator().get_risk_level("005930.KS")
```
